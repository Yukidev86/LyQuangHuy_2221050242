HTML5 Syntax

HTML documents should start with the declaration.

The HTML element has a lang attribute:

< html lang="en" > ...

All HTML elements must be properly nested:


<div>
    <p>Hello</p>
</div>

Most elements have an opening and closing tag:


<p>This is a paragraph.</p>

Attribute values should always be quoted:


<img src="img.jpg" alt="image">
