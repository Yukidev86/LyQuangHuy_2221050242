HTML vs XHTML

XHTML là phiên bản HTML tuân theo chuẩn XML (strict).

Ví dụ tài liệu XHTML:


<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE html
    PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
    <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
    <title>XHTML Example</title>
</head>
<body>

<h1>Hello World</h1>
<img src="image.jpg" alt="Example" />

</body>
</html>

Quy tắc chính trong XHTML:

  • Tất cả thẻ phải đóng đầy đủ.
  • Thuộc tính phải có giá trị trong dấu ngoặc kép.
  • Thẻ tự đóng phải dùng dấu />.
  • Sử dụng namespace xmlns cho thẻ <html>.
